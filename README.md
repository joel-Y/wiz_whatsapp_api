# wiz_whatsapp_api# 

Serverless API connecting Gemini AI Studio to Odoo CRM.

## Setup

1. Deploy to Vercel
2. Add environment variables:
   - ODOO_URL
   - ODOO_DB
   - ODOO_USERNAME
   - ODOO_PASSWORD

## API Endpoint

POST /api/odoo

## Actions

- search_customer
- create_lead
- search_products
- list_leads
